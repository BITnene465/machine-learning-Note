本仓库主要用于存tex的笔记文件

主要构架如下:

```
MyMLNotes/
|-- main.tex        # 主LaTeX文件，可能包括导言区、章节等
|-- chapters/       # 存放各个章节的LaTeX文件
|   |-- chapter1.tex
|   |-- chapter2.tex
|   |-- ...
|-- images/         # 存放项目中使用的图像文件
|-- bibliography/   # 存放参考文献文件
|-- styles/         # 存放自定义的样式文件（如果有的话）
|-- .git/           # Git仓库的元数据，不要手动修改
|-- .gitignore      # 指定Git忽略的文件和目录
|-- README.md       # 项目的说明文档
```
